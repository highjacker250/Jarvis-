# Jarvis AI Assistant

Personal AI voice assistant for Android.
Supports Hindi, English, Bhojpuri.

## Setup
1. Android Studio mein open karo
2. OpenRouter.ai se free API key lo
3. Settings mein key paste karo
4. Run karo

## Features
- Voice commands Hindi/English
- App control, calls, SMS
- WiFi, Bluetooth, flashlight control
- Face recognition login
- Works offline also
